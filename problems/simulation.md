# シミュレーション（Simulation）

## 問題1: ロボットの最終位置

ロボットが原点 `(0, 0)` にいる。命令文字列 `commands`（`U`, `D`, `L`, `R`）に従って移動したとき、最終座標を求めよ。

### 入力例
```
commands = "UURRDDLL"
```

### 出力例
```
(0, 0)
```

---

## 問題2: じゃんけんトーナメント

N 人の参加者の手（`R`, `S`, `P`）が配列で与えられる。トーナメント形式（隣同士で対戦、勝者が次に進む）で優勝者の番号を求めよ。N は 2 の冪とする。

### 入力例
```
hands = ["R", "S", "R", "P"]
```

### 出力例
```
0
```
