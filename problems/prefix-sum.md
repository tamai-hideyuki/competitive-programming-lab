# 累積和（Prefix Sum）

## 問題1: 区間和クエリ

整数配列 `nums` と Q 個のクエリ `(l, r)` が与えられる。各クエリに対して `nums[l] + ... + nums[r]` を求めよ。

### 入力例
```
nums = [1, 2, 3, 4, 5]
queries = [(1, 3), (0, 4)]
```

### 出力例
```
9
15
```

---

## 問題2: 合計がKになる部分配列の数

整数配列 `nums` と整数 `K` が与えられる。連続する部分配列の和が `K` になるものの個数を求めよ。

### 入力例
```
nums = [1, 1, 1]
K = 2
```

### 出力例
```
2
```
