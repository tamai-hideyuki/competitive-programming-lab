# セグメント木（Segment Tree）

## 問題1: 区間最小値クエリ（RMQ）

整数配列 `A` に対して以下のクエリを処理せよ。
- `update(i, x)`: `A[i]` を `x` に更新
- `query(l, r)`: `A[l..r]` の最小値を返す

### 入力例
```
A = [5, 3, 7, 9, 1, 4]
query(1, 4) → 1
update(4, 10)
query(1, 4) → 3
```

## 問題2: 区間加算・区間和（遅延伝播）

整数配列 `A` に対して以下のクエリを処理せよ。
- `add(l, r, x)`: `A[l..r]` の各要素に `x` を加算
- `sum(l, r)`: `A[l..r]` の合計を返す

### 入力例
```
A = [1, 2, 3, 4, 5]
sum(0, 4) → 15
add(1, 3, 10)
sum(0, 4) → 45
```
