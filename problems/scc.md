# 強連結成分分解（SCC）

## 問題1: 強連結成分の数

N 頂点 M 辺の有向グラフが与えられる。強連結成分の数を求めよ。

### 入力例
```
N = 5
edges = [(0,1), (1,2), (2,0), (3,4)]
```

### 出力例
```
3 ({0,1,2}, {3}, {4})
```

---

## 問題2: 2-SAT

N 個のブール変数と M 個の節（各節は 2 リテラルの OR）が与えられる。充足可能か判定せよ。

### 入力例
```
N = 2
clauses = [(x1 ∨ x2), (¬x1 ∨ x2), (¬x2 ∨ x1)]
```

### 出力例
```
true (x1 = true, x2 = true)
```
