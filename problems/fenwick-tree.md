# Fenwick Tree（Binary Indexed Tree）

## 問題1: 点更新・区間和

整数配列 `A` に対して以下のクエリを処理せよ。
- `add(i, x)`: `A[i]` に `x` を加算
- `sum(l, r)`: `A[l..r]` の合計を返す

### 入力例
```
A = [1, 2, 3, 4, 5]
sum(0, 2) → 6
add(1, 5)
sum(0, 2) → 11
```

## 問題2: 転倒数

配列 `A`（要素は 1〜N の順列）の転倒数を BIT を使って `O(N log N)` で求めよ。

### 入力例
```
A = [3, 1, 4, 2]
```

### 出力例
```
3
```
