# 計算幾何（Computational Geometry）

## 問題1: 凸包

2次元平面上に N 個の点が与えられる。凸包を構成する点を反時計回りに出力せよ。

### 入力例
```
points = [(0,0), (1,1), (2,0), (1,2), (0,2)]
```

### 出力例
```
[(0,0), (2,0), (1,2), (0,2)]
```
