# 貪欲法（Greedy）

## 問題1: 区間スケジューリング

N 個の仕事があり、それぞれ開始時刻と終了時刻が与えられる。重ならないように選べる仕事の最大数を求めよ。

### 入力例
```
jobs = [(1, 3), (2, 5), (4, 7), (6, 8)]
```

### 出力例
```
3
```

---

## 問題2: コイン問題

金額 `amount` が与えられる。硬貨の種類 `[1, 5, 10, 50, 100, 500]` を使って、最小枚数でおつりを作れ。

### 入力例
```
amount = 1234
```

### 出力例
```
15
```
