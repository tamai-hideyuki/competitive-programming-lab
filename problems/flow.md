# フローアルゴリズム（Flow Algorithms）

## 問題1: 最大流

N 頂点の有向グラフに容量が設定されている。ソース `s` からシンク `t` への最大流を求めよ。

### 入力例
```
N = 4
edges = [(0,1,10), (0,2,5), (1,3,5), (2,3,10), (1,2,3)]
s = 0, t = 3
```

### 出力例
```
15
```

## 問題2: 二部マッチング

N 人の労働者と M 個の仕事があり、各労働者が担当できる仕事のリストが与えられる。最大何人に仕事を割り当てられるか求めよ。

### 入力例
```
N = 3, M = 3
worker 0: [0, 1]
worker 1: [1, 2]
worker 2: [2]
```

### 出力例
```
3
```
