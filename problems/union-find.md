# Union-Find（Disjoint Set Union）

## 問題1: 友達の友達

N 人の人がいて、M 組の友人関係が与えられる。人 `a` と人 `b` が（間接的にでも）友達かどうか判定せよ。

### 入力例
```
N = 5
relations = [(0, 1), (1, 2), (3, 4)]
query: 0 と 2 → true
query: 0 と 3 → false
```

---

## 問題2: 連結成分の数

N 頂点 M 辺の無向グラフが与えられる。連結成分の数を求めよ。

### 入力例
```
N = 5
edges = [(0, 1), (1, 2), (3, 4)]
```

### 出力例
```
2
```
