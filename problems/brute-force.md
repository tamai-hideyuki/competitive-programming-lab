# 全探索（Brute Force）

## 問題1: 二つの数の合計

配列 `nums` と整数 `target` が与えられる。`nums[i] + nums[j] == target` となる `(i, j)` の組を求めよ。

### 入力例
```
nums = [2, 7, 11, 15]
target = 9
```

### 出力例
```
(0, 1)
```

## 問題2: 3つの数の最大積

整数配列 `nums`（長さ N >= 3）が与えられる。任意の3つの要素の積の最大値を求めよ。

### 入力例
```
nums = [-10, -10, 5, 2]
```

### 出力例
```
500
```
