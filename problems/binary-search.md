# 二分探索（Binary Search）

## 問題1: ソート済み配列の探索

ソート済み整数配列 `nums` と整数 `target` が与えられる。`target` のインデックスを返せ。存在しなければ `-1` を返せ。

### 入力例
```
nums = [1, 3, 5, 7, 9]
target = 5
```

### 出力例
```
2
```

---

## 問題2: 答えで二分探索

N 本のケーブル（各長さ `L[i]`）から、同じ長さの K 本のケーブルを切り出したい。切り出せるケーブルの最大長を求めよ。

### 入力例
```
L = [8, 15, 20]
K = 6
```

### 出力例
```
7
```
