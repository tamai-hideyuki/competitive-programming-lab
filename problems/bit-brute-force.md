# bit全探索（Bit Brute Force）

## 問題1: 部分和問題

N 個の整数からなる配列 `A` と整数 `S` が与えられる。部分集合の和が `S` になるものが存在するか判定せよ。

### 入力例
```
A = [1, 2, 4, 8]
S = 13
```

### 出力例
```
true (8 + 4 + 1 = 13)
```

---

## 問題2: 最大XOR

N 個の非負整数からなる配列 `A` が与えられる。部分集合の XOR の最大値を求めよ。

### 入力例
```
A = [3, 5, 6]
```

### 出力例
```
7 (3 XOR 5 XOR 6 = 0 なので、5 XOR 6 = 3、3 XOR 6 = 5、3 XOR 5 = 6、3 = 3、5 = 5、6 = 6 → 最大は 6)
```
